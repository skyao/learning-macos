### 介绍

System Toolkit是Mac平台上的一款系统工具。System Toolkit提供了系统维护、隐私保护、[磁盘清理](http://www.ucbug.com/z/cipanql/)、电池维护、网络状态监控、网络流量监控等实用的功能。

通过 System Toolkit 可以随时监控到进程负载，CPU温度，主内存消耗，存储空间，磁盘活动，网络接口上的通信等。
